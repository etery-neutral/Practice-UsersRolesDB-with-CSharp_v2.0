# Инструкция для проверяющего: запуск проекта "UsersRoles"

Перед запуском убедитесь, что на компьютере установлены:

.NET 8.0 SDK;  
PostgreSQL 17;  
Visual Studio 2022;   
  
  
Шаг 1. Скачивание проекта  
Склонируйте репозиторий или скачайте ZIP-архив:  
(Для упрощения проверки лучше после скачивания архива извлечь архив UsersRoles v2.0.rar)  
  
Шаг 2. Создание базы данных  
-> Запустите pgAdmin  
-> В левой панели разверните Servers → PostgreSQL XX.  
-> Введите пароль пользователя postgres, который вы задали при установке PostgreSQL.  
-> Нажмите правой кнопкой мыши на Databases → Create → Database.  
-> Назовите базу данных (можно любое имя, главное указать его потом в строке подключения).  
-> Нажмите Save.  
  
Шаг 3. Запуск проекта  
-> Запуск через Visual Studio  
-> Откройте файл решения UsersRoles.sln в Visual Studio 2022.  
-> Дождитесь завершения загрузки и восстановления NuGet-пакетов (происходит автоматически).  
-> Убедитесь, что в Обозревателе решений файл appsettings.json присутствует и его свойство "Копировать в выходной каталог" установлено в "Копировать, если новее".  
-> (Замените данные файла appsettings.json на актуальные (см. файл appsettings.example.json)  
-> Нажмите F5 или кнопку "Пуск" (зелёный треугольник) для запуска.  
  
Всё готово. (Необходимые пояснения есть в коде.)
